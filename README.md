## 🚀 Featured: Halley

**[Halley](https://github.com/A-Wattamwar/halley) — Your production traffic is your test suite.**

An open-source, self-hostable **LLM observability and regression testing platform**. It records production AI agent runs as bit-fidelity cassettes, turns any run into a permanent regression test, and replays your whole fixture library in CI at $0 LLM cost, then bisects to the exact commit that broke it.

- ⚙️ **Rust** ingester (tokio, axum, tonic), **OpenTelemetry**-native, normalizes 5 trace dialects into one canonical schema
- 📊 **Next.js 14** dashboard · **TypeScript** & **Python** SDKs · `halley ci` CLI for replay and bisect
- 🗄️ Tiered storage: **ClickHouse** + **PostgreSQL** + **Redis Streams**
- 📈 **4,792 spans/sec** sustained · **0% error rate** · 1.44M spans ingested with **zero data loss**
- 🐳 **Docker Compose** local · **Kubernetes** Helm for production · MIT licensed
